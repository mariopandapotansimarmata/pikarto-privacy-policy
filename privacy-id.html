Kebijakan Privasi

Pikarto (“kami”, “aplikasi”, atau “App”) menghargai privasi Anda. Kebijakan Privasi ini menjelaskan bagaimana kami menangani informasi Anda saat menggunakan aplikasi kami.

1. Informasi yang Kami Kumpulkan

  • Input Mikrofon: Kami menggunakan mikrofon perangkat Anda untuk merekam suara selama sesi latihan.
  • Transkrip & Evaluasi: Rekaman audio dikirim ke API pihak ketiga (OpenAI untuk transkripsi dan Gemini API untuk evaluasi). Hasil transkrip dan umpan balik evaluasi disimpan secara lokal di perangkat Anda sebagai riwayat pembelajaran.

2. Bagaimana Kami Menggunakan Informasi

  • Transkripsi: Suara Anda diproses oleh API OpenAI untuk menghasilkan transkrip teks.
  • Evaluasi: Transkrip dikirim ke Gemini API untuk memberikan umpan balik terkait Struktur, Efektivitas, dan Relevansi.
  • Riwayat Pembelajaran: Hanya transkrip dan hasil evaluasi yang disimpan secara lokal di perangkat Anda untuk membantu melacak perkembangan Anda.

3. Penyimpanan & Retensi Data

  • Data Audio: Kami tidak menyimpan atau mempertahankan rekaman audio Anda.
  • Transkrip & Evaluasi: Disimpan hanya secara lokal di perangkat Anda. Anda dapat menghapusnya kapan saja dengan menghapus data aplikasi.
  • Tidak Ada Penyimpanan Cloud: Kami tidak mengunggah atau menyimpan data pribadi Anda di server kami.

4. Berbagi Data

Kami tidak menjual, membagikan, atau mengungkapkan data pribadi Anda kepada pihak ketiga. Satu-satunya data yang dikirim ke luar adalah audio (untuk transkripsi) dan transkrip (untuk evaluasi), yang diproses secara aman oleh API pihak ketiga terpercaya (OpenAI dan Gemini).

5. Keamanan

Kami mengambil langkah yang wajar untuk melindungi informasi Anda. Namun, karena kami bergantung pada API pihak ketiga, praktik privasi mereka juga berlaku saat memproses data Anda. Silakan tinjau kebijakan privasi OpenAI dan Gemini untuk detail lebih lanjut.

6. Hak Anda

  • Anda dapat menghapus transkrip dan riwayat evaluasi kapan saja.
  • Anda juga dapat menghapus instalasi aplikasi untuk menghapus seluruh informasi yang tersimpan.

7. Privasi Anak-Anak

Aplikasi ini tidak ditujukan untuk anak-anak di bawah usia 13 tahun. Kami tidak secara sadar mengumpulkan data dari anak-anak.

8. Perubahan Kebijakan

Kami dapat memperbarui Kebijakan Privasi ini dari waktu ke waktu. Perubahan akan ditandai dengan “Tanggal Berlaku” yang baru.
